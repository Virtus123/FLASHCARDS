Programação

O que é uma variável em programação?

Programação

Um espaço na memória para armazenar dados que podem mudar durante a execução do programa.

Programação

Qual a diferença entre uma função e um método?

Programação

Função é um bloco de código independente; método é uma função associada a um objeto.

Programação

O que é um loop "for"?

Programação

Uma estrutura de repetição que executa um bloco de código um número definido de vezes.
